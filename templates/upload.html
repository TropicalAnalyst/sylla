CSV Format Requirements:

The CSV file should have the following columns:

  • title - Source title (required)
  • content - Source content
  • category - Source category
  • keep - Set to "true" to keep the source (optional)
  • ip - IP address (optional)
  • ip_description - Description for IP (optional)
  • url - URL (optional)
  • url_description - Description for URL (optional)
  • hash_value - Hash value (optional)
  • hash_type - Hash type (md5, sha1, sha256, etc.) (optional)
  • hash_description - Description for hash (optional)
Example CSV Format:
title,content,category,keep,ip,ip_description,url,url_description,hash_value,hash_type,hash_description
"Malicious IP","Found in spam logs","Threat Intel",true,"192.168.1.1","Spam source","http://example.com","Malicious domain","d41d8cd98f00b204e9800998ecf8427e","md5","Malicious file hash"
