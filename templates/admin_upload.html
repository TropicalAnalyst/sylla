CSV Format Requirements

The CSV file should have one row per IOC. Sources with the same title will be automatically grouped.

Required Columns:
  • title: The title of the source.
  • content: The content/description for the source.
  • ioc_type: The type of the IOC (e.g., ip, url, hash, domain).
  • ioc_value: The value of the IOC.
Optional Columns:
  • category: The category for the source (defaults to 'General').
  • keep: Set to 'true' or '1' to prevent the source from being deleted by retention policies.
  • ioc_tags: Comma-separated tags for the individual IOC.
Example:
title content ioc_type ioc_value category ioc_tags
Phishing Campaign Analysis Analyzed a recent phishing campaign. ip 192.168.1.100 Phishing campaign-alpha,c2
Phishing Campaign Analysis Analyzed a recent phishing campaign. url http://evil-site.com/login.php Phishing phishing-kit,login
Malware Sample #45B Reverse engineered malware sample. hash ... Malware dropper,emotet
